    Finding the traces…

    A trace is the point where a line (or its extension) meets a reference plane.

    HT (Horizontal Trace): where the line meets HP. Extend the front view to XY at h, then project down to the top view.

    VT (Vertical Trace): where the line meets VP. Extend the top view to XY at v, then project up to the front view.

    • HT = Horizontal Trace
    • VT = Vertical Trace
    • h = front view meets XY
    • v = top view meets XY
    • ⊥ perpendicular projector
    • dashed = construction / extension

    Rules: ∥ to a plane → no trace on it · ⟂ to a plane → trace coincides with the projection point · h and v always lie on XY.

    When a line is inclined to both planes, neither projection shows the true length. The Rotating-Line method swings each view parallel to XY to reveal it.

    Part A rotates the top view → True Length & θ (with HP). Part B rotates the front view → True Length & φ (with VP).
